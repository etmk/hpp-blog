毎回 Dock とかに置いてあるアイコンをダブルクリックするのは面倒なので。

open コマンドを使うとイケる。

```bash
open /Applications/Google\ Chrome.app/
```

エイリアスを貼っておくと便利。

```bash

alias chrome='open /Applications/Google\ Chrome.app/'
```
