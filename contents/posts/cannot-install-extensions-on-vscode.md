# VSCodeのアップデートはしていますか

Proxyしている訳でも無いのに拡張機能がインストール出来ないときはだいたいこれです。
とりあえずアップートが出来ないか確認してください。

Macなら左上のメニューから"Check for Updates"か、または"Restart to Update"でどうにかなる可能性が高いです。
